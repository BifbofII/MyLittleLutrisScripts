Gamepad profiles for Future Pinball

FP supports natively gamepad, but if you prefer like use triggers as flippers, you will find here two gamepad profiles.

Depend your gamepad, choose appropriate app below, install it on your Linux distribution, import profile and start game, you can now play with your gamepad.

x360 controller profile to use with antimicro --> fp_x360.gamecontroller.amgp

You can download Antimicro here: https://github.com/AntiMicro/antimicro

Steam controller profile to use with sc-controller --> fp_steam.sccprofile

You can download sc-controller here --> https://github.com/kozec/sc-controller

You will find gamepad profiles in your Lutris fp wineprefix: c:\Future Pinball\gamepad profiles\
